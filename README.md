#  Projeto CRUD

Aplicação web feita em Angular para a disciplina de WEB I do curso de Especialização em Desenvolvimento Ágil de Software- UFPR.

## Descrição

Projeto contendo um CRUD de Aluno e um de Curso.

### Dados de Aluno:
- id – numérico
- nome – string
- cpf – string
- e-mail – string
- data de nascimento – string
### Dados de Curso:
- id – numérico
- nome – string
- link - string
